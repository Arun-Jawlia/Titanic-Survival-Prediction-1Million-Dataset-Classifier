# Titanic Survival Prediction Classifier with 1 Million Dataset

Dataset Name: Titanic Huge Dataset - 1M Passengers
Dataset Link: https://www.kaggle.com/datasets/marcpaulo/titanic-huge-dataset-1m-passengers

## Command to Follow project
1. Create a folder , checkout to that folder
1. conda create -p venv python==3.11
2. conda activate ./venv
3. Create requirements.txt file
4. pip install -r requirements.txt
5. pip install ipykernel
